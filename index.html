Sobre mim

Sou estudante de desenvolvimento front-end, com planos de me tornar uma desenvolvedora full stack. Desde criança, sempre fui curiosa e criativa, o que despertou minha paixão por tecnologia. Meu principal objetivo é construir soluções que resolvam problemas reais por meio da tecnologia.

Projetos

Projeto: Android

Projeto: Cordel Moderno

Este foi meu projeto mais recente, inspirado na literatura de cordel. A proposta foi criar uma página com um visual estilizado que unisse tradição e tecnologia. Utilizei HTML e CSS para desenvolver uma estrutura limpa, responsiva e com destaque para o efeito de parallax scrolling, que dá profundidade ao conteúdo à medida que o usuário rola a página. Foi um exercício criativo que reforçou meu aprendizado em posicionamento de elementos e efeitos visuais modernos. ver

Projeto: Check-list

Este projeto foi desenvolvido para facilitar a organização das tarefas diárias, oferecendo uma ferramenta simples e funcional para adicionar, marcar como concluídas e remover atividades. A interface foi construída com HTML5, CSS3 e JavaScript, seguindo boas práticas de semântica e responsividade. O design minimalista garante uma experiência limpa e intuitiva, permitindo ao usuário focar no que realmente importa.
